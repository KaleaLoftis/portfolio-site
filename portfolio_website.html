Project Overview

Over the span of 8 weeks I built a personal portfolio website from scratch to show my web develpment skills that I have acquired so far in my educational carrer. This project was a personal challenge to gain more hands-on experience in web development. I used Behance and Dribble to research and get design inspiration. I made simple UX/UI mockups using Figma. Through the utilization of HTML, l structured the website's content, including sections for an introduction, project showcases, skills, and a way to contact me. Using CSS I created a visually appealing and responsive design across various devices.
